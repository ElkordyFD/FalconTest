# 🦅 Falcon - Swag Labs Automation Suite

**Falcon** is a robust Selenium automation framework designed to validate critical functionalities of the [Swag Labs](https://www.saucedemo.com) web application. Built with Java and Maven, Falcon follows the Page Object Model (POM) and enables scalable, maintainable, and reusable automated tests.

---

## 🚀 Features

- 🔐 Login: Valid & invalid credential tests
- 🛒 Products: Add/remove items from cart
- 📦 Checkout: Complete order flow validation
- 🔍 Filters: Product sorting by price/name
- 📋 Menu: Navigation and reset functionality
- 📎 Footer: Social media links and external validations
